Accessibility Statement

Monkey Got Thumbs is committed to making our website accessible to everyone, including people with disabilities. We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA.

What We Do

  • Semantic HTML throughout — proper heading hierarchy, landmarks, and structure
  • Keyboard navigation support on all interactive elements
  • Skip-to-content links on every page
  • ARIA labels and roles where needed
  • Focus indicators that meet contrast requirements
  • Support for prefers-reduced-motion and prefers-contrast media queries
  • Alt text on all images
  • Sufficient colour contrast ratios for text and interactive elements
  • Responsive design from 320px to widescreen

Testing

We test accessibility using a combination of automated tools (Lighthouse, axe) and manual testing with keyboard navigation and screen readers (VoiceOver on macOS).

Known Issues

If you encounter any accessibility barriers on this site, please let us know. We treat accessibility issues as high-priority bugs.

Feedback

If you have difficulty accessing any content on this site, please contact us at hello@monkey-got-thumbs.com. We will respond within 2 business days and work to resolve the issue.
