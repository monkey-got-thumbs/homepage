The short version

AI right now is basically a few companies offering tools that can read, write, analyse, and generate things. You talk to them in plain English. They talk back. That's it.

The big ones

ChatGPT By OpenAI. The one most people have heard of. Free tier available. Good all-rounder.
Claude By Anthropic. Tends to be more careful and detailed. Good for longer work. Free tier available.
Gemini By Google. Built into Google's ecosystem. Good if you already live in Google Docs and Gmail.
Copilot By Microsoft. Built into Windows and Office. Good if you already use Microsoft tools.

Try it now

Go to chatgpt.com or claude.ai. Sign up for free. Type something you'd normally Google.

Instead of a list of links, you get an answer. Sometimes it's great. Sometimes it's wrong. That difference is the whole game.

Three things to know going in

It makes things up. Confidently. It will state fiction as fact with zero hesitation. Always check anything that matters.

It's a tool, not an oracle. It works best when you give it a clear task and review the result. Worst when you trust it blindly.

You're in charge. It doesn't decide anything. You ask, it responds, you decide what to do with it.
